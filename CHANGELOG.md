# Changelog / История изменений

All notable changes to this project will be documented in this file.

Все заметные изменения в проекте будут фиксироваться в этом файле.

## [0.1.0] - 2026-06-17

### Added

- Initial public MVP release of `knowdiff`
- CLI commands for `add`, `list`, `check`, `remove`, and `export`
- SQLite persistence for tracked pages, snapshots, and diffs
- HTML extraction, section parsing, and deterministic diffing
- Markdown export for the latest recorded diff
- Test suite, static analysis configuration, and GitHub Actions CI

### Добавлено

- Первый публичный MVP-релиз `knowdiff`
- CLI-команды `add`, `list`, `check`, `remove` и `export`
- SQLite-хранилище для страниц, снимков и diff-отчётов
- Извлечение текста из HTML, парсинг секций и детерминированное сравнение
- Экспорт последнего diff-отчёта в Markdown
- Набор тестов, конфигурация статических проверок и GitHub Actions CI
