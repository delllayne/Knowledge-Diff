# Security Policy / Политика безопасности

## English

### Supported versions

Knowledge Diff currently supports the latest `0.1.x` line.

### Reporting a vulnerability

Please do not open a public issue for a suspected security problem.

Instead:

1. Contact the maintainer privately through GitHub.
2. Include reproduction steps and affected versions.
3. If possible, include a minimal proof of concept.

We will aim to acknowledge the report within 7 days and share a remediation plan after triage.

## Русский

### Поддерживаемые версии

Сейчас поддерживается актуальная ветка `0.1.x`.

### Как сообщить об уязвимости

Пожалуйста, не публикуйте предполагаемую проблему безопасности в открытом issue.

Вместо этого:

1. Свяжитесь с сопровождающим приватно через GitHub.
2. Приложите шаги воспроизведения и затронутые версии.
3. По возможности добавьте минимальный proof of concept.

Мы постараемся подтвердить получение сообщения в течение 7 дней и после триажа сообщить план исправления.
